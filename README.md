# Discord ↔ Telegram бот управления

Простая, но мощная связка из двух ботов, которая позволяет **управлять участниками Discord-сервера прямо из Telegram**.

С помощью этого проекта администратор может:
- Получить список всех участников сервера (кроме ботов)
- Кикнуть пользователя
- Забанить пользователя
- Пингануть пользователя в указанном канале

Всё управление — через удобные инлайн-кнопки в Telegram-боте. Никаких команд в Discord писать не нужно.

## Как это работает

Discord-бот запущен с повышенными правами и запускает локальный HTTP API на `localhost:8000`.  
Telegram-бот обращается к этому API и выполняет действия от имени Discord-бота.

## Функции

- `/start` → Выводит приветственное сообщение и кнопку "список пользователей"
- Выбор пользователя -> кнопки «Заблокировать / Кикнуть / Пингануть»  

## Требования

- Python 3.9+
- Discord-сервер, где бот имеет права Administrator (Ban Members, Kick Members, Send Messages)
- Telegram-бот

## Установка и запуск
- Скопируйте код
- Установите зависимости
- Запустите ботов командами python discord_bot.py и python telegram_bot.py
